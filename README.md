## Install:
```bash
> pkg update && pkg upgrade -y
> pkg install git
> pkg install nodejs
> git clone https://github.com/KazeDevID/get-pic-waifuuu
> cd get-pic-waifuuu
> npm install
```

### Run
```bash
> node index.js
```
